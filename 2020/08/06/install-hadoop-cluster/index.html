  1. 选择新建虚拟机

  4. 选择 Linux，Centos 7 64位
  5. 命名虚拟机 hadoop1
  6. 使用默认 20G 磁盘大小
  7. 不修改虚拟机硬件配置，完成创建虚拟机

  10. 选择 CD/DVD，使用 ISO 映像文件，选择 ISO 文件位置
